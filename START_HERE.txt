⚡ QUICK START (3 STEPS)
═══════════════════════════════════════════════════════════════════════════════

Terminal 1 - Start Backend:
  cd backend
  npm install
  cp .env.example .env
  node seed.js
  npm run dev

Terminal 2 - Start Frontend:
  cd frontend
  npm install
  npm start

Browser:
  Open http://localhost:3000
  Login: admin@bloodlink.com / admin123


🐳 DOCKER (1 COMMAND)
═══════════════════════════════════════════════════════════════════════════════

docker-compose up

Then visit: http://localhost:3000
